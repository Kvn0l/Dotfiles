# Hyprland X Qogir

Basic and compact Hyprland rice built around [Qogir-Dark](https://github.com/vinceliuice/Qogir-theme) GTK Theme

![Inspired by Qogir](.config/hypr/wall.png)

|Wallpaper:         | [Hyprpaper](https://github.com/hyprwm/hyprpaper)                                             |
